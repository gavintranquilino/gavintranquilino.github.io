About

GuhBot is a Discord bot developed to enhance your server experience. With a wide range of commands and features, GuhBot helps streamline moderation tasks while adding fun elements to your community.

Developed by Gavin Tranquilino, GuhBot is regularly maintained and updated with new features based on community feedback.

Features

  • Moderation tools to manage your community
  • Fun commands to engage your members
  • Utility functions for server management
  • Custom configurations for your specific needs
  • Regular updates with new features

Commands

Moderation

  • !kick @user - Kick a user from the server
  • !ban @user - Ban a user from the server
  • !mute @user - Mute a user in the server

Fun

  • !meme - Shows a random meme
  • !guh - Responds with a "guh" message
  • !8ball - Ask the magic 8-ball a question

Utility

  • !help - Show all available commands
  • !info - Show bot and server information
  • !ping - Check bot response time
