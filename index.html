What I do

Python Programming

Python is my main programming language. I use Python for scripting bots using APIs such as Discord, Twitter, YouTube, etc. Python was the first text-based programming language that I used. I programmed apps that would help me solve my math homework, bots using selenium to scrape memes off the internet, and I even created a bot to generate views on YouTube for my friends. I started programming in Scratch to make games and animations for school. I knew that there are much more powerful programming languages that can do much more than to make simple games and animations using a drag and drop system such as Scratch. With the help of some tutorials, articles, and Stack Overflow, I was able to teach myself how to use Python for online projects, or even for daily use.

HTML, CSS, JAVASCRIPT

This website was fully written in HTML, CSS, and JavaScript. I love to create websites and host them using GitHub Pages, and I am exploring how to use custom domains to publish my websites on other domain names. With these tools, I was able to create a website for my custom Discord bot, my portfolio, a personalized website that I show my class, and even a few website portals for my friends to link their social media. I knew that programs written in Python and Scratch would be difficult to share with others. I discovered that I could make my own websites to display and showcase my code or even make websites to be able to share my passion for learning and programming to the world through something as simple as a link.

Arduino and Raspberry Pi

With an Arduino and a Raspberry Pi, I am able to create a bridge between software and hardware. With a microcontroller such as the Arduino UNO, I have created a motion sensor and custom LED light control, using the Arduino programming langauge that is based off of C++. While I am still exploring how to use this technology, I am eager to explore the capabilites of this device. With my Raspberry Pi, I am able to host my Discord bot, and create a cheap web server for my other websites. I have explored the software of the Raspberry Pi using the Raspberry Pi Operating System, which based on Debian Linux. I was able to connect my desktop to my Raspberry Pi through Secure Shell in order to upload files and start my webserver wirelessly.
