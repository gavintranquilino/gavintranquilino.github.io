What I do

Python Programming

Python is my main programming language. From scripting to local webservers for home automation, I use Python on the regular. My proficiency in Python has been mostly attributed to self-directed learning, enough to compete in the 2023 Canada Computing Contest. I have always had a thing for programming, especially as it makes sense logically. I find satisfaction in being able to create a program that runs reliably, and is scalable. Much of my digital literacy and proficiency, and research skills I attribute to programming as a whole. My favorite project with Python is definitely my Discord bot for general purposes seen below.

3D printing and CAD

Software aside, I bought myself a 3d printer as a Christmas gift for myself in 2022. (Probably the best gift I've gotten). Since then, I've been able to learn CAD software such as Fusion360 and SolidWorks (contribution in FIRST Robotics Team 2702). I have used 3D printing to create characters, gifts, and stamps. One of my dreams is to be proficient in creating, so that I can combine mechanical and electronics design to create gadgets. I find it satisfying whenever I can birth my ideas into the real world through CAD design. Probably my favorite 3D printing project is my crude home light switch automation. More on it below.

Arduino and Raspberry Pi

With an Arduino and a Raspberry Pi, I am able to create a bridge between software and hardware. With a microcontroller such as the Arduino UNO, I have created a motion sensor and custom LED light control, using the Arduino programming langauge that is based off of C++. While I am still exploring how to use this technology, I am eager to explore the capabilites of this device. With my Raspberry Pi, I am able to host my Discord bot, and create a cheap web server for my other websites. I have explored the software of the Raspberry Pi using the Raspberry Pi Operating System, which based on Debian Linux. I was able to connect my desktop to my Raspberry Pi through Secure Shell in order to upload files and start my webserver wirelessly.

Mechatronics Engineering Student

I am a highschool student, who wants to know a little about everything. My main goal is to expand my knowledge on new technologies. I always wanted to know how to program and work with computers, so I waited and waited, hoping that I would soon learn how to code. I did not realize that there were so many free and online tutorials and resources that I can use to teach myself how to code. I did not want to wait for my school to teach me, if I could do it myself.

I challenged myself to teach myself the skills that I wanted to learn. I taught myself how to make games on Scratch, make bots with Python, CAD printables, etc. By having the initiative and patience to teach myself the skills I wanted to know, I was able to let my creativity spill, and work on projects that I wanted to work on. I learned that I did had the capability to teach myself these new concepts, and how I could apply them to my daily life.
