Using Fusion360, I created a 3D model of his pyramid Vtuber character design. The model is available right now on Thingiverse.

I had it print in two parts with mouse ears on the sharp corners of the model to prevent the need for bed adhesion. I created holes with a slight tolerance on the base to allow for the glasses to print separately and fit snugly into the base without glue.
